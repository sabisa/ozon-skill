## 三、利润计算公式

### 3.1 基础参数

```
汇率: 1₽ = 0.08¥ (即 1¥ ≈ 12.5₽)
Ozon佣金: 约 12% (儿童玩具类)
物流成本: 约 280₽/件 (2kg以内)
```

### 3.2 利润计算

```python
# 输入
selling_price = 2000  # 售价(₽)
cost_yuan = 35  # 采购成本(¥)
logistics = 280  # 物流(₽)
commission_rate = 0.12  # 佣金率

# 计算
commission = selling_price * commission_rate  # 216₽
revenue_after_commission = selling_price - commission  # 1784₽
cost_rub = cost_yuan / 0.08  # 437.5₽
profit_rub = revenue_after_commission - logistics - cost_rub  # 1066.5₽
profit_yuan = profit_rub * 0.08  # 85.3¥
profit_margin = profit_rub / selling_price  # 53.3%

print(f"利润: {profit_rub:.0f}₽ ≈ {profit_yuan:.0f}¥ (利润率: {profit_margin:.1%})")
```

### 3.3 快速对照表

| 售价(₽) | 佣金(12%) | 物流(₽) | 采购(¥) | 利润(₽) | 利润(¥) | 利润率 |
|---------|----------|---------|---------|---------|---------|--------|
| 1000 | -120 | -250 | 15 | 205 | 16 | 20.5% |
| 1500 | -180 | -280 | 25 | 465 | 37 | 31.0% |
| 1800 | -216 | -280 | 30 | 704 | 56 | 39.1% |
| 2000 | -240 | -280 | 35 | 785 | 63 | 39.3% |
| 2500 | -300 | -320 | 45 | 1030 | 82 | 41.2% |
| 3000 | -360 | -350 | 55 | 1275 | 102 | 42.5% |

**目标：利润 ≥ 30¥，利润率 ≥ 25%**

## 四、蓝海商品筛选流程

### 4.1 第一步：市场初筛

```
搜索关键词 → 查看搜索结果数量
           ↓
      结果数 < 50? → 是 → 进入下一步
           ↓ 否
      竞争太激烈，考虑其他品类
```

### 4.2 第二步：头部商品分析

```
滚动加载TOP20商品 → 记录评论数和评分
                   ↓
            评论数 < 5000?
            评分 4.0-4.6?
                   ↓ 是
            进入商品详情分析
```

### 4.3 第三步：商品详情分析（布丁猫）

```
打开商品详情页 → 激活布丁猫插件 → 记录数据
                                           ↓
         ┌─────────────────────────────────┐
         │ 上架时间: 3个月-2年 ✓            │
         │ 近30天销量: 100-500 ✓            │
         │ 评论数: 50-2000 ✓                │
         │ 评分: 3.8-4.5 ✓                 │
         │ 无认证要求? ✓                    │
         └─────────────────────────────────┘
                      ↓ 全部满足
              记录为候选商品
```

**重点关注的潜力股**：
- ⭐ 月销 100-300 + 评论 < 100 + 上架 < 6个月 → 蓝海机会
- ⭐ 月销 200-500 + 评分 3.8-4.2 + 有差评共性 → 改进机会
- ⭐ 月销 300-500 + 上架 1-2年 + 评论 500-1500 → 稳定需求

### 4.4 第四步：差评分析

```
查看商品评论 → 找出差评共性
             ↓
      是否有共性问题?
      (如: 太小了/质量差/图片不符)
             ↓ 是
      找到差异化改进机会 ✓
             ↓ 否
      商品同质化严重，考虑其他
```

## 五、选品决策矩阵

### 5.1 评分标准（满分100分）

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 市场容量 | 20% | 月销100-500: 20分, 50-100: 15分, <50: 5分 |
| 竞争程度 | 25% | 评论50-500: 25分, 500-2000: 15分, >2000: 5分 |
| 上架时间 | 15% | 3个月-1年: 15分, 1-2年: 10分, <3月或>2年: 5分 |
| 利润空间 | 25% | 利润>80¥: 25分, 30-80¥: 15分, <30¥: 5分 |
| 认证难度 | 15% | 无认证: 15分, 简单认证: 10分, 复杂认证: 0分 |

### 5.2 决策阈值

| 总分 | 决策 | 建议 |
|------|------|------|
| ≥ 75分 | 强烈推荐 | 立即测试 |
| 55-74分 | 推荐 | 可以测试 |
| 35-54分 | 谨慎 | 小批量测试 |
| < 35分 | 不推荐 | 避开 |
