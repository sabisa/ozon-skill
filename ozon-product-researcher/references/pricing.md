# Ozon 定价与利润公式

## 基础参数

| 项目 | 数值 |
|------|------|
| 汇率 | 1 ₽ ≈ 0.08 ¥ |
| 佣金率 | 8-15%（儿童玩具约12%） |
| 物流成本 | 250-350 ₽/件 |

### 物流成本参考

| 重量范围 | 物流费（约） |
|---------|------------|
| 0-0.5 kg | 180-220 ₽ |
| 0.5-1 kg | 220-280 ₽ |
| 1-2 kg | 280-350 ₽ |
| 2-5 kg | 350-500 ₽ |
| 5+ kg | 500+ ₽ |

## 利润计算公式

### 标准公式

```
净利润(₽) = 售价 - 佣金 - 物流费 - 采购成本
净利润(¥) = 净利润(₽) × 汇率

其中：
- 佣金 = 售价 × 12%
- 采购成本(₽) = 采购成本(¥) / 汇率
```

### 快速估算

```
净利润(¥) ≈ (售价 × 0.88 - 物流费) × 0.08 - 采购成本(¥)
```

## 采购成本上限计算

**目标：净利润 ≥ 40¥**

```
采购成本上限(¥) = (售价 × 0.88 - 物流费) × 0.08 - 40
```

## 定价策略

### 1. 竞争定价
```
定价 = 市场均价 × (1 - 折扣比例)
```

### 2. 利润定价
```
定价 = (目标利润 + 采购成本 + 物流费) / (1 - 佣金比例)
```

### 3. 折扣策略

| 折扣 | 效果 | 建议 |
|------|------|------|
| 0-20% | 轻微 | 新品期 |
| 20-50% | 中等 | 常规促销 |
| 50-80% | 大幅 | 清仓甩卖 |
| 80%+ | 极端 | 谨慎使用 |

---

## 成本优化建议

### 1. 降低采购成本
- 批量采购
- 直接对接工厂
- 1688 货源比价

### 2. 降低物流成本
- 使用 FBS 头程优惠
- 合理打包减少体积
- 批量发货

### 3. 优化佣金
- 选择佣金较低品类
- 使用 Ozon 官方物流
- 参与推广活动
