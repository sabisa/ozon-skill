# Ozon 定价与利润公式

## 基础参数

### 汇率
| 货币 | 汇率 | 更新时间 |
|------|------|---------|
| 1 ₽ (卢布) | ≈ 0.08 ¥ (人民币) | 实时 |
| 1 $ (美元) | ≈ 7.2 ¥ | 实时 |
| 1 € (欧元) | ≈ 7.9 ¥ | 实时 |

### Ozon 佣金费率
| 品类 | 佣金比例 | 说明 |
|------|---------|------|
| 电子产品 | 8-12% | 含手机配件 |
| 儿童玩具 | 8-15% | 含玩具、游戏 |

## 成本计算公式

### 1. Ozon 佣金
```
佣金 = 售价 × 佣金比例
```

### 2. 物流成本（FBS 模式）

| 重量范围 | 物流费（约） |
|---------|------------|
| 0-0.5 kg | 180-220 ₽ |
| 0.5-1 kg | 220-280 ₽ |
| 1-2 kg | 280-350 ₽ |
| 2-5 kg | 350-500 ₽ |
| 5+ kg | 500+ ₽ |

### 3. 仓储费
```
仓储费 = 按月计算
- 首月免费
- 超期仓储费: 10-50 ₽/月
```

### 4. 最终利润公式

```
净利润(₽) = 售价 - 佣金 - 物流费 - 仓储费 - 采购成本
净利润(¥) = 净利润(₽) × 汇率
```

## 简化利润计算

### 快速估算公式

```
净利润(¥) ≈ (售价 × 0.85 - 280 - 采购成本) × 0.08
```

其中：
- 0.85 = 扣除佣金后的比例（假设12%佣金）
- 280 = 估算物流费
- 0.08 = 汇率

## 利润达标判断

### 目标利润 ≥ 50 ¥

```javascript
// 达到50¥利润所需的最低售价
最低售价 = (目标利润 / 汇率 + 物流费) / (1 - 佣金比例)
最低售价 = (50 / 0.08 + 280) / 0.88
最低售价 ≈ 1,250 ₽
```

## 定价策略

### 1. 竞争定价
```
定价 = 市场均价 × (1 - 折扣比例)
```

### 2. 利润定价
```
定价 = (目标利润 + 采购成本 + 物流费) / (1 - 佣金比例)
```

### 3. 折扣策略

| 折扣 | 效果 | 建议 |
|------|------|------|
| 0-20% | 轻微 | 新品期 |
| 20-50% | 中等 | 常规促销 |
| 50-80% | 大幅 | 清仓甩卖 |
| 80%+ | 极端 | 谨慎使用 |

## 成本优化建议

### 1. 降低采购成本
- 批量采购
- 直接对接工厂
- 1688 货源比价

### 2. 降低物流成本
- 使用 FBS 头程优惠
- 合理打包减少体积
- 批量发货

### 3. 优化佣金
- 选择佣金较低品类
- 使用 Ozon 官方物流
- 参与推广活动
