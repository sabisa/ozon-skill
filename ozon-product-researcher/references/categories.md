# Ozon 品类参考指南

## 📊 选品核心数据指标（新手友好版）

### 蓝海 vs 红海判断标准

| 指标 | 蓝海（推荐） | 红海（避开） |
|------|-------------|-------------|
| 搜索结果数 | < 100 | > 300 |
| 头部商品评论数 | < 10000 | > 20000 |
| 新品比例 | > 20% | < 5% |
| 月销 TOP 商品 | 100-500 件 | < 50 或 > 2000 件 |
| 上架时间 | 3个月-2年 | > 3 年 |
| 评论数 | 50-2000 | > 5000 |
| 评分 | 3.8-4.5 | > 4.7 |

### 利润参考指标

| 指标 | 参考值 |
|------|--------|
| 汇率 | 1₽ ≈ 0.08¥ |
| 佣金率 | 8-15%（儿童玩具约12%） |
| 物流成本 | 250-350₽/件 |
| 目标利润率 | > 25% |
| 目标净利润 | ≥ 30¥/件 |

### 重点潜力股特征

- ⭐ 月销 100-300 + 评论 < 100 + 上架 < 6个月 → 典型蓝海
- ⭐ 月销 200-500 + 评分 3.8-4.2 + 有差评共性 → 改进机会
- ⭐ 月销 300-500 + 上架 1-2年 + 评论 500-1500 → 稳定需求
