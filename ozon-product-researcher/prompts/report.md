# 报告生成提示词

```json
{
  "research_info": {
    "category": "儿童玩具",
    "price_min": 1500,
    "price_max": 3000,
    "country": "China",
    "date": "2026-07-07"
  },
  "products": [
    {
      "name": "磁性钓鱼玩具",
      "price": 1723,
      "original_price": 3000,
      "reviews": 5666,
      "rating": 4.9,
      "discount": "42%",
      "url": "https://www.ozon.ru/product/...",
      "keywords": ["магнитная рыбалка", "детская рыбалка", "развивающая игрушка"]
    }
  ]
}
```

## 报告结构模板

### 1. 执行摘要
简要说明研究发现和核心建议（3-5句话）

### 2. 筛选条件
| 条件 | 标准 |
|------|------|
| 品类 | {{category}} |
| 价格区间 | {{price_min}} - {{price_max}} ₽ |
| 原产国 | {{country}} |
| 目标利润 | ≥ {{profit_target}} ¥ |

### 3. 蓝海选品总览
品类排名表格，包含：排名、品类、竞争度、利润空间、推荐指数

### 4. 品类详情
每个品类2-3个推荐商品，包含：
- 商品名称、链接、售价
- 评论数、评分、上架时间
- **关键词**（Seerfar SKU Reverse 获取）
- 采购成本上限
- 利润计算

### 5. 关键词分析
每个商品的搜索关键词汇总，用于：
- 标题优化
- 广告投放
- 竞争分析

### 6. 利润达标汇总表
所有商品的采购成本上限汇总

### 7. 试单建议
首批测试商品清单（数量、预估采购价、预估利润）

### 8. 风险提示
- 认证要求
- 物流成本
- 竞争风险
- 汇率波动

### 9. 商品链接与关键词汇总
所有推荐商品的链接和关键词汇总表

---

## 格式要求

- 使用 Markdown 格式
- 包含表格展示数据
- 使用 emoji 增强可读性
- 价格使用 ₽ 符号
- 链接不带追踪参数（去掉 ?at=... 部分）
- 中文输出

## 核心指标参考

### 商品评估标准

| 指标 | 优秀 | 合格 | 差 |
|------|------|------|-----|
| 月销 | 100-500 | 50-100 或 500-1000 | < 50 或 > 2000 |
| 上架时间 | 3个月-1年 | 1-2年或<3个月 | > 3年 |
| 评论数 | 50-500 | 500-2000 | > 5000 |
| 评分 | 3.8-4.3 | 3.5-3.8 或 4.3-4.7 | < 3.5 或 > 4.7 |

### 蓝海潜力股特征

```
⭐ 月销 100-300 + 评论 < 100 + 上架 < 6个月 → 典型蓝海
⭐ 月销 200-500 + 评分 3.8-4.2 + 有差评共性 → 改进机会
⭐ 月销 300-500 + 上架 1-2年 + 评论 500-1500 → 稳定需求
```

### 利润计算

```
采购成本上限(¥) = (售价 × 0.88 - 物流费) × 0.08 - 目标利润(40¥)

示例：
- 售价 2000₽，物流 280₽
- 采购成本上限 = (2000 × 0.88 - 280) × 0.08 - 40 = 56¥
```

## 关键词采集说明

Seerfar SKU Reverse 模块使用方法：

1. 打开商品详情页
2. 点击 Seerfar 插件图标
3. 选择「SKU Reverse」功能
4. 输入或自动识别商品 SKU
5. 点击「搜索」获取关键词
6. 等待 5-10 秒数据加载
7. 截图保存关键词数据

### 关键词格式示例

```markdown
**关键词**（Seerfar SKU Reverse）：
- магнитная рыбалка（搜索量：高，竞争度：中）
- детская рыбалка（搜索量：中，竞争度：低）
- развивающая игрушка（搜索量：中，竞争度：低）
- игрушка для детей 3+（搜索量：低，竞争度：低）
- рыбалка магнитная детская（搜索量：低，竞争度：低）

**应用建议**：
- 标题优化：主要使用前两个高搜索量关键词
- 广告投放：使用前三个关键词进行推广
- 市场定位：该商品主要定位为"儿童益智玩具"
```
