# OZON 物流价格计算器

快速查询布丁猫 OZON 数据平台的 GUOO 物流价格。

## 使用方法

直接输入商品信息即可，格式示例：

```
查询 GUOO 物流：家园类目, 采购成本30, 重量1000g, 包装400x500x600
```

或者：

```
类目[家园], 成本[30], 重量[1000], 体积[400x500x600]
```

## 参数说明

| 参数 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 类目 | ✅ | OZON 商品类目名称 | 家园、住宅和花园、服装等 |
| 采购成本 | ✅ | 商品采购成本(¥) | 30 |
| 重量 | ✅ | 商品重量(克) | 1000 |
| 体积 | ✅ | 包装尺寸(长x宽x高 mm) | 400x500x600 |

## 默认计算参数

| 参数 | 默认值 |
|------|--------|
| 毛利率 | 35% |
| 国内运费 | ¥5 |
| 划线折扣 | 50% |
| 退货率 | 5% |
| 其他费用 | 3% |
| 发货类型 | FBS |
| 取件方式 | 取货点自提 |
| 收货地区 | 俄罗斯 |

## 执行流程

使用 playwright-cli skill 执行以下步骤：

1. `playwright-cli open https://www.bdmozon.com/analysis/calculator --browser=chrome`
2. 点击类目选择区域
3. 搜索并选择指定类目
4. 填写采购成本、重量、包装体积
5. 点击"计算结果"按钮
6. 等待计算完成
7. 在筛选框输入"GUOO"并搜索
8. 解析结果返回给用户

## 输出格式

返回 GUOO 物流的三种运输方式：

| 运输方式 | 时效 | 售价(¥/₽) | 物流费用 | 总成本 | 毛利值 | 是否计抛 |
|----------|------|-----------|----------|--------|--------|----------|
| Express Small | 5-14天 | ¥228.93/₽2583 | ¥63.44 | ¥98.44 | ¥80.13 | 不计抛 |
| Economy Small | 12-24天 | ¥180.56/₽2037 | ¥42.64 | ¥77.64 | ¥63.20 | 不计抛 |
| Standard Small | 7-19天 | ¥204.74/₽2310 | ¥53.04 | ¥88.04 | ¥71.66 | 不计抛 |

## 示例调用

用户输入：`家园类目, 采购成本30, 重量1000g, 体积400x500x600`

Claude 执行：
```bash
playwright-cli open https://www.bdmozon.com/analysis/calculator --browser=chrome
# ... 自动化操作 ...
playwright-cli snapshot
# 解析并返回 GUOO 物流价格结果
```

## 注意事项

- 需要用户已登录布丁猫账号
- 如果未登录，先引导用户完成登录
- 物流价格会根据实际数据实时计算
- GUOO 有三种运输方式：Express(快速)、Economy(经济)、Standard(标准)
